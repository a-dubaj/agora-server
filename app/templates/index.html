This is the Agora v0.5.1.

This site is very much under construction, but feel free to look around:
  • {{nodes}} lists all nodes in this Agora; a node is the set of all subnodes with a given title, or otherwise about the same entity. Subnodes can come from a variety of sources.
  • {{subnodes}} lists all subnodes in this Agora; currently these are mostly notes as volunteered by users.
  • {{journals}} displays all journal entries (these are notes matching YYYY-MM-DD).
  • {{users}} displays all users in this Agora, click through to see their subnodes.
  • {{search}} offers full text search over all subnodes (supports regexes).
The [[wikilink]] is the heart of the Agora. /node/foo will render every subnode that resolves to wikilink [[foo]]. For example: [[Agora]].

For more information, please visit the Agora repository. If you're interested in knowing what's coming, please refer to Agora plan. {% endblock %}
